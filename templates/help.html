Using the Site

This section explains how to use the site. If you need further help, please contact us.

Account Management

You can change your password by clicking on your username in the navigation bar while logged in. Once your password is changed, you will be logged out of all active sessions.

There is currently no way to delete an account. Contact the administrators if needed.

Upload Guidelines

In order to maintain a clean and navigable site, all uploaded content must meet the following criteria.

What to upload?

Users are prohibited from uploading pornographic, explicit, offensive, or illegal content. All images must comply with copyright laws. Violations will result in removal of the content and revocation of upload access.

Supported File Types

The following file types are supported:

  • .apng
  • .png
  • .jpg
  • .jpeg
  • .gif
  • .avif

Uploads of any other file type will be rejected.

Deleting Files

You can only delete images you have uploaded yourself. To see how this works via the API, visit the API Documentation.
