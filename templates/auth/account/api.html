Getting started

Upload a file from the shell — the response carries the canonical link:

curl -H "Authorization: $KLS_TOKEN" \
     -F "file=@screenshot.png" \
     {{ api_base }}/images/upload

Every endpoint, request shape, and scope is documented at /api/docs.
